# 產品願景與目標

## 產品概述

**企業永續性評估平台** 是一個數據驅動的企業分析工具，專門用於評估企業的六大核心能力並進行多維度比較分析。

## 核心價值主張

提供專業、即時的企業永續性評估，透過數據視覺化與多維度分析，輔助投資決策者與企業管理層做出明智的商業決策。

## 目標用戶

### 主要用戶群
1. **企業分析師** - 需要快速評估企業整體表現
2. **投資決策者** - 基於數據進行投資組合決策
3. **永續發展專業人員** - 追蹤企業 ESG 績效
4. **企業管理層** - 監控公司營運績效與競爭力

### 用戶需求
- 快速獲取企業多維度評估結果
- 直觀的數據視覺化呈現
- 歷史趨勢追蹤與同業比較
- 可靠的評分計算邏輯

## 核心功能

### 六大評估維度
1. **營運能力** - 存貨週轉率、應收帳款週轉率、總資產週轉率
2. **財務能力** - ROE、ROA、負債比率等財務指標
3. **未來力** - 成長潛力與發展趨勢評估
4. **AI數位力** - 數位轉型與科技應用程度
5. **ESG永續力** - 環境、社會、治理績效
6. **創新能力** - 研發投入與創新成果

### 主要功能模組
- **雷達圖多維度比較** - 同時展示六大維度評分
- **公司間績效比較** - 選擇多家公司進行對比分析
- **歷史趨勢追蹤** - 時間序列數據分析
- **數據視覺化儀表板** - 卡片式數據呈現
- **評分計算引擎** - 基於商業邏輯的自動評分
- **數據管理功能** - 手動輸入與維護企業財務數據

## 產品限制與約束

### 當前限制
- 數據需手動維護與更新
- 僅支援 GitHub Pages 公開部署
- 無用戶認證與權限管理

### 未來考量
- 需規劃企業內部部署架構
- 需設計登入與權限系統
- 需準備資料庫遷移方案 (Supabase → SQL Server)

## 設計原則

### 視覺設計
- **淺色主題**: Slate 色系為主的專業介面設計
- **卡片化設計**: 資訊分組清晰，提升可讀性
- **色彩編碼**: 績效評級用顏色直觀表示 (優異/良好/一般/待改善/風險)
- **響應式布局**: 適應不同裝置螢幕尺寸

### 數據視覺化原則
- 多維度雷達圖為核心視覺元素
- 趨勢折線圖追蹤時間序列變化
- 進度條與評分卡片直觀呈現績效
- 色彩系統提供即時狀態反饋

### 用戶體驗原則
- 快速回應 (< 1 秒載入)
- 清晰的視覺層次
- 直觀的操作流程
- 最小化認知負荷

## 關鍵決策紀錄

### 為何選擇前端計算評分？
- 減少後端運算負擔
- 快速響應用戶互動
- 便於調整評分邏輯

### 為何使用 Supabase？
- 快速開發原型
- 免費方案適合初期測試
- 未來可遷移至 SQL Server

### 為何採用 GitHub Pages 部署？
- 零成本部署方案
- 適合初期展示與測試
- 未來會遷移至企業內部
